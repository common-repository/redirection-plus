== Description ==

301 redirection is very important when you change an old url structure to a new url structure. Your audience will have 404 errors and also your keyword ranking will be lost in search engines. To avoid those issues you need to use 301 redirect for your to keep safe your search engine ranking as well as your audience experience. After setting 301 redirect within a few days, the search engine will re-index your new url without hurting your website ranking. Redirection Plus comes with very fast redirection and hit analytics so you can easily detect the number of redirections based on user date country and browser.

Redirection Plus is a simple redirection plugin you can set 301, 302 and 307 easily with this plugin and also you can track how many hits you are getting for each redirection.

COUNTRY BASED USER REDIRECTION
This plugin comes with another advanced feature you can filter out visitors based on country and also can send to other destinations based on their country. you also can check the analytics number of hits by country, browser and also by date.

1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Does 301 redirection is seo redirection? =
301 redirection is best practice to redirect any URL without losing keyword ranking.

= What is 302 redirection? =
302 redirection means temporary redirection. The best practice is to use 301 redirection.
